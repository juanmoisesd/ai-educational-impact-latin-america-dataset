# Codebook

This codebook describes the variables in the dataset.

## Variables

### participant_id
- **Description:** Unique identifier for each participant.
- **Type:** String (Anonymized)

### country
- **Description:** Country where the data was collected.
- **Type:** Categorical (e.g., Mexico, Colombia, Argentina, etc.)

### academic_performance
- **Description:** Standardized score of academic performance.
- **Type:** Numeric (0-100)

### motivation_score
- **Description:** Measured student motivation level.
- **Type:** Numeric (1-5 Likert scale)

### self_regulation_score
- **Description:** Measured self-regulation of learning.
- **Type:** Numeric (1-5 Likert scale)
