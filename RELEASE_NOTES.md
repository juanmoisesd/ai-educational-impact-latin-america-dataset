# Release Notes - v1.0.0

We are pleased to announce the first official release of the **Impact of Artificial Intelligence on Educational Processes in Latin America** dataset repository.

## Highlights
- **Comprehensive Metadata:** Multiple interoperable formats including DataCite, schema.org, and RO-Crate.
- **Reproducibility:** Ready-to-use Docker environment and analysis notebooks.
- **Security:** Cryptographic manifests and security policies included.
- **Ethics:** Detailed ethical guidelines and data protection protocols.

## What's New
This is the initial version, establishing a robust framework for open science and data sharing in the field of AI in education.
